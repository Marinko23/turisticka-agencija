# Pokretanje:

### Prvi put:
```bash
pip install Django 
pip install pandas
pip install Pillow 
pip install django-extensions
pip install pydotplus
pip install django-mathfilters
pip install requests
```

### Sa instaliranim bibliotekama:
```bash
python manage.py migrate
```

### Testiranje:
```bash
python manage.py test
```

### Konacno pokretanje:
```bash
python manage.py runserver
```
